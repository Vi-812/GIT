Конечные точки API, доступные для запроса к Git Check Alive


Ваше приложение может выполнять запросы к следующим конечным точкам REST:
https://isgitalive.karo-dev.ru/api/repo - получение статистики по репозиторию
https://isgitalive.karo-dev.ru/api/issues-statistic - получение статистики по issues
https://isgitalive.karo-dev.ru/api/full - получение всей статистики


Для GET запросов доступны следующие параметры:
name - имя репозитория в формате 'https://github.com/Vi-812/git_check_alive' или 'vi-812/git_check_alive'
skipCache - игнорирование базы данных, только свежие данные (False по умолчанию)
token - токен, можно передать в headers запроса


Для POST запросов доступны следующие параметры в теле запроса:
name - имя репозитория в формате 'https://github.com/Vi-812/git_check_alive' или 'vi-812/git_check_alive'
skipCache - игнорирование базы данных, только свежие данные (False по умолчанию)
token - токен
